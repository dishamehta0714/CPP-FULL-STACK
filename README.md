Railway Reservation System (C++)
Project Description

This project is a Railway Reservation System developed using C++.
The system allows users to add train records, display all trains, and search for a train by its number.

It demonstrates the use of Object-Oriented Programming (OOP) concepts such as:

Classes and Objects
Constructors and Destructors
Static Data Members
Encapsulation
Vector (STL)
Features
Add New Train Record
User can enter train details.
Display All Train Records
Shows all trains stored in the system.
Search Train by Number
User can search for a train using its train number.
Menu Driven Program
Easy to use menu interface.
Train Details Stored

The following information is stored for each train:

Train Number
Train Name
Source
Destination
Train Time
Technologies Used
C++
Standard Template Library (STL)
Vector
Object-Oriented Programming
OOP Concepts Used
Class

Train and RailwaySystem classes are used to organize the program.

Constructors
Default constructor initializes train details.
Destructor
Destructor updates the static train counter.
Static Member

trainCount keeps track of the number of train objects.

Encapsulation

Train details are kept private and accessed through functions.

Program Menu
--- Railway Reservation System Menu ---

1. Add New Train Record
2. Display All Train Records
3. Search Train by Number
4. Exit
How the Program Works
User selects an option from the menu.
If option 1 is selected, the system asks for train details.
If option 2 is selected, all train records are displayed.
If option 3 is selected, the system searches for a train using its number.
Option 4 exits the program.
